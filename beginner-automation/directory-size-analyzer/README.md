# 📂 Directory Size Analyzer

A Python automation tool that analyzes the total size of a directory and its subdirectories.  
This script helps users identify how much disk space a folder consumes.

---

## 🚀 Features

- Calculates total size of a directory
- Scans all subdirectories automatically
- Displays folder sizes in MB
- Simple command-line interface
- Useful for disk usage monitoring

---

## 🛠️ Technologies Used

- Python 3
- os module
