# Folder Cleaner

A simple Python automation script to organize and clean up files in a folder.

## Features

- Automatically sorts files by type into subfolders
- Deletes the empty folders
- Customizable file extensions and folder names

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/folder-cleaner.git
   ```
2. Navigate to the project directory:
   ```bash
   cd folder-cleaner
   ```
3. Run the script:
   ```bash
   python folder_cleaner.py
   ```

## Requirements

- Python 3.x

## Configuration

Edit `folder_cleaner.py` to customize file types and destination folders.
